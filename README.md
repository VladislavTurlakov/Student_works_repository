Данный проект был реализован на втором курсе университета в 2023 году.  

Описание проекта:  
Приложение, представляющее собой графическую программу для просмотра, поиска, добавления, редактирования и удаления студенческих работ, хранящихся в субд SQLite.  

Технологический стек:  
Python, PyQt5, SQLite
